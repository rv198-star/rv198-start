# KiU v0.6.4 Source Hygiene Addendum And Long-Term Deep Reading Plan

Date: `2026-04-25`

## Purpose

This addendum records a small v0.6.4 follow-up after the Shiji and Mao anthology runs.

The generated and usage layers scored well, but the source layer remained lower on thick historical, biographical, and argument-heavy materials. The interpretation is straightforward: KiU's presentation and distillation are already useful, but the upstream source substrate still struggles with very complex books.

## v0.6.4 Low-ROI-Cost Fixes

These fixes stay inside v0.6.4 because they are source hygiene improvements, not a new version direction.

- Navigation markdown such as `SUMMARY.md` and `README.md` is excluded from directory ingestion.
- `source_shape.source_file_count` is computed after `source_files` is populated, so directory corpora report real file counts.
- Degenerate pure-number headings such as `# 一` and `## 二` are ignored as semantic section anchors.
- Existing graph extraction remains broad; this does not suppress source evidence, it only prevents navigation and numbering artifacts from polluting source structure.

## Long-Term High-Difficulty Plan

The high-difficulty solution is not part of this v0.6.4 patch. It should be planned as a later source-depth track.

Long-term candidates:

- whole-book reading pass before local extraction
- cross-chapter theme and mechanism clustering
- case mechanism evidence packs with multiple independent anchors
- argument-line reconstruction across chapters
- counter-case and abuse-case mining before skill distillation
- LLM-assisted source map review with explicit provenance and cost controls

## Decision

v0.6.4 closes the low-cost hygiene gap only. The deeper source-understanding work should be scheduled later because it is closer to a major extraction architecture upgrade than a release patch.
