# v0.5.1 Usage Expression Rebuild Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rebuild KiU's usage-facing expression layer so generated skills carry scenario-specific trigger, non-trigger, edge, and refusal families strongly enough to close the remaining same-scenario gap without boundary drift.

**Architecture:** Keep KiU's thick audited spec intact, but add a structured `scenario family` layer under each generated skill and synthesize a shorter usage-facing expression from it into the published markdown. Phase 1 only rebuilds scenario families and usage expression plumbing; Phase 2 revisits the `value-assessment` vs `margin-of-safety-sizing` topology only if fresh evidence still shows a structural gap after Phase 1.

**Tech Stack:** Python 3, `unittest`, existing `src/kiu_pipeline/*` generation pipeline, YAML bundle artifacts, `scripts/run_book_pipeline.py`, `scripts/benchmark_reference_pack.py`.

---

## Scope

This plan intentionally separates two problems:

- Phase 1: `scenario family layer + usage expression synthesis`
- Phase 2: `value-assessment` topology correction

Only Phase 1 starts immediately. Phase 2 is conditional on fresh benchmark evidence after Phase 1.

## Phase 2 Trigger Condition

Only start Phase 2 if fresh post-Phase-1 evidence still shows:

- `margin-of-safety-sizing` remains the dominant negative pair
- the residual gap is structural rather than wording-only

If triggered, Phase 2 will:

- add a KiU-native `value-assessment` parent skill
- keep `margin-of-safety-sizing` as a narrower specialization
- rerun alignment and benchmark scoring without relaxing the workflow-vs-agentic boundary

## Exit Rule

Do not claim `v0.5.1` closure just because:

- all cases pass
- the artifact lead remains positive
- the usage delta becomes only “small”

The release line is only honestly closed when fresh evidence shows:

- same-scenario average usage delta is no longer negative
- pass-rate parity or better is preserved
- no improvement came from boundary drift
