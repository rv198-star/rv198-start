## User-Facing Findings

### 1. Financial Statement now passes the user-facing coverage gate

The first user-facing review found that Financial Statement was structurally fixed but still narrow: only one thick action skill plus a workflow gateway. The rerun recovers `accounting-quality-signal-check` as a second thick action skill while keeping deterministic workflows routed out.

The visible skill set is still not full financial-analysis coverage, but it now covers two distinct user-value surfaces: business-value anchoring and accounting-quality risk gating. `coverage_fit` improves from `78` to `86`.

### 2. Gateway skills are useful, but should stay visibly second-class

`workflow-gateway` artifacts help users route deterministic material without polluting thick skills. They are valuable as boundary infrastructure, not as the main user value. The rerun makes this visible inside `SKILL.md` by saying the gateway is a thin workflow router, not a thick judgment skill.

### 3. Poor Charlie remains the strongest user-visible skill set

Poor Charlie's six skills read like mature user-facing mental models: they have intuitive names, clear decision situations, and direct action or calibration value. This is the cleanest example of what a high user-facing score should look like.

### 4. Shiji and Mao now expose the source-to-mechanism bridge more clearly

Historical and political-theory skills already scored well on boundary clarity and action helpfulness. The rerun improves source readability by adding an explicit mechanism chain and teaching the mechanism-density model to recognize classical-history action/constraint/consequence language. In Shiji, the displayed mechanism-density anchor for the three skills rises to `0.875`.

### 5. The user-facing view proved useful as an optimization driver

The first user-facing review averaged `90.0` and exposed two model-level gaps. After model-level fixes, the rerun averages `91.6`, matching the C-class average without manually editing generated outputs. This supports keeping user-facing dimensions as a standard release review surface.
