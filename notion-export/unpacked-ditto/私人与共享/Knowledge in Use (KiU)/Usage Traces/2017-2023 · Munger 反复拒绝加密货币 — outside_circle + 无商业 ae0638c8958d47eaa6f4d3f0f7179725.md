# 2017-2023 · Munger 反复拒绝加密货币 — outside_circle + 无商业性

Anchors: node://berkshire-agm/2018/qa-crypto; node://munger-djco/2023/crypto-rejection
Available Info: (1) Munger 无法理解加密货币的内在价值来源;(2) 无现金流生成机制;(3) 价格由情绪和投机驱动、无企业对应的生产己力。
Decision: 公开反复拒绝,称之为 "rat poison" / "artificial gold" (2018 Berkshire AGM + 2021 + 2022 + 2023 Munger 致股东信)。Berkshire 不配置。
Decision Date: 2018年5月5日
Options Considered: 1) 小比例配置对冲(富達 Druckenmiller 走过的路);
2) 表态中立;
3) 公开彻底拒绝。
Outcome: 避开 2018 崩盘(BTC 从 $20K 跌到 $3.2K、-84%)和 2022 崩盘(从 $69K 跌到 $15.8K、-77%)。FTX/灰度/LUNA 等一系列机构倡鞭。决策长期正收益。
Outcome Known: Yes
Rationale: 本 skill 在这里触发了双重拒绝:
(a) verdict = outside_circle (无商业模式可理解)
(b) 进一步: 判断标的本身不构成"生意"(no cashflow-generating business) → 超越 circle-of-competence 范围 → 在 rationale 里记为 "meta-rejection"
(c) recommended_action = decline + 对外表达明确
Retrospective: boundary test 的教科书案例 — skill 的 boundary 层发现"标的不构成生意"时应升级拒绝,而不仅是"不够了解"。这里揭示 schema 的一个演化方向: 未来可添加 fails_hard_http://when.target_has_no_cashflow_business 作为 meta-level boundary。
Situation: 2017 年起加密货币市场暴涨(BTC 从 $1K 涨至 $69K 峰值)。媒体、机构、零售反复追问 Munger 和 Buffett 观点。
