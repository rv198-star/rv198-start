## v0.1 · MVP 范围 (当前重心)

### 范围内

- 单 corpus:《穷查理宝典》
- **手工构造** 5 个 skill (不跑 pipeline,红通线手拉)
- 验证 `use` schema 的完备性
- 验证 KiU Test 三问的有效性
- 构建 scenario_corpus: 30 个人工编写的投资决策情境
- 构建 evaluation_corpus: 20 个有已知结果的真实决策
    - 来源: 巴菲特公开持仓变动历史 + 短革库报告 + 公开访谈
- 所有 5 个 skill 在 evaluation 上跑走

### 范围外 (v0.1 不做)

- ✖️ 自动 pipeline
- ✖️ 跨 corpus union
- ✖️ MCP server
- ✖️ 多 domain profile (只做 `default` + `investing`)
- ✖️ Live execution feedback

### 成功标准

1. **全部 5 个 skill 通过 KiU Test 三问**
2. **每个 skill 挂载 ≥3 个 usage traces** (结构完整的决策轨迹)
3. **每个 skill 在 evaluation gate 上达标** (按 [The KiU Test · 三问与外部评测](The%20KiU%20Test%20%C2%B7%20%E4%B8%89%E9%97%AE%E4%B8%8E%E5%A4%96%E9%83%A8%E8%AF%84%E6%B5%8B%2042e1662a734a4e93ac1d7a7b727171ce.md) 阈值)
4. **盲评不劣于 cangjie-skill 的 12-skill 版本**

### 候选 5 个 skill (穷查理宝典)

1. `circle-of-competence` —— 能力圈识别
2. `invert-the-problem` —— 逆向思考
3. `margin-of-safety-sizing` —— 安全边际下的仓位制定
4. `bias-self-audit` —— 二十五种认知偏误的自检
5. `opportunity-cost-of-the-next-best-idea` —— 以次优选项为弃留阈值
