# Archive Boundary

`_archive/` contains historical imports and frozen background material that remains useful for traceability.

It is not a current entry point for understanding KiU / 学以致用. Start with `README.md`, `docs/README.md`, `evidence/README.md`, or `review-pack/README.md` instead.

Archive contents are not used by the default generation pipeline. They must not be treated as hidden upstream source material for current generated skills.

Current source materials and reviewer-facing generated artifacts live under `review-pack/current/`. Current methodology and architecture explanations live under `docs/`.
