# Generation Workflow

Use KiU generation when the user wants to convert a source book or source bundle into action skills.

Default workflow:

1. Confirm the source material and output root.
2. Run the KiU pipeline rather than editing generated skills by hand.
3. Review source, generated, usage, and practical-effect scores.
4. Keep generated artifacts traceable to source evidence and pipeline provenance.
5. Record evidence boundaries in the review pack.
